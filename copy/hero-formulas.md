# Hero Formulas

Use these as starting structures, not final copy.

## Outcome + audience + mechanism

```text
Get [outcome] for [audience] without [pain], using [mechanism].
```

## Category ownership

```text
The [category] for [audience] who need [specific high-value outcome].
```

## Speed / certainty

```text
[Outcome] in [timeframe] — with [proof/risk reversal].
```

## Premium service

```text
A calmer, sharper way to [solve expensive problem].
```

## Local/trust service

```text
[Service] done clearly, quickly, and without surprises.
```

## Subcopy pattern

```text
We help [audience] do [job] by [process/mechanism]. You get [deliverable/result], [trust point], and [next step].
```

## CTA examples

Good:

- Get a quote
- Book a 15-minute call
- Send project brief
- Check availability
- Start with references
- Calculate price
- Upload photos

Avoid:

- Learn more
- Discover
- Unlock growth
- Get started, unless the next step is truly self-serve
