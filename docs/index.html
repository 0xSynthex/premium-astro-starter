FAQ

Use the sections as a system, not as a template pile.

Can this starter be used for client work?

Yes. Replace placeholder copy, use real proof/assets, and keep private client material out of public git.

Does it require paid tooling?

No. The baseline uses Astro, Playwright, GitHub Pages, ImageMagick, and cwebp.

When should sections be deleted?

Delete any section that does not clarify the offer, prove trust, answer an objection, or move the user toward the CTA.
